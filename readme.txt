KN Social Slide is a free WordPress Plugin that lets your visitors know about your facebook, twitter, google+, feedburner and random entry!

== Description ==

* We make setting up a simple slide to your wordpress and lets your visitors know about your facebook, twitter, feedburner email subscriptions and google+ and random post from your own wordpress. Don't worry, you can turn on and off, re-position and re-order any slide you don't want to anytime.

== Installation ==

= Installation =

1. Upload the file `kn-social-slide.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Please change the settings and enjoy!

= Updating =

If you had a trouble in updating this plugin, try deactive old version and delete the folder `kn-social-slide` from the `/wp-content/plugins/` directory. Then, try download new version and install as usual.

== Frequently Asked Questions ==

= The slide doesn't work? =

* Try to turn on/off the jQuery.noconflict at setting page

= The slide still doesn't work after turn on/off jQuery.noconflict? =

* May have conflict with other plugins or template. Try deactive all plugin, then detect which plugin cause this problem by try active them 1 by 1. If the slide still not working, change the template!
